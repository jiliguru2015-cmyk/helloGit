# v73.20-rag-core-spec.md

## 项目目标

构建企业级本地部署 RAG Agent Core。

技术栈：

* FastAPI
* Qdrant
* Ollama
* qwen2.5
* nomic-embed-text
* React + Vite
* SSE Streaming
